# AES-XGCM.NET

A .NET implementation of [AES-XGCM](https://soatok.blog/2022/12/21/extending-the-aes-gcm-nonce-without-nightmare-fuel/) using [ECB instead of CBC-MAC](https://old.reddit.com/r/crypto/comments/zry2uh/extending_the_aesgcm_nonce_without_nightmare_fuel/j16985b/).
